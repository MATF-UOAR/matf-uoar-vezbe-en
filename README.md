# UOAR – exercise materials

This repository contains materials for the practical part of the course **"Introduction to Computer Organization and Architecture"**, taught in the first year of the I (Computer Science) track at the Faculty of Mathematics.

The materials cover the fundamental concepts of assembly programming for the `x86-64` and `ARM64` processor architectures.

The materials include examples, exercises, and accompanying code used during the lab sessions.

The repository is under active development, so any suggestions, corrections, and comments are welcome.
