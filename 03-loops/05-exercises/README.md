# Self-study mini-examples

This unit contains five short, ready-made mini-examples for self-study after the third week.

The idea is not to introduce entirely new concepts, but to reinforce:

- the `while`, `for`, and `do-while` styles of loops
- working with the multiplication and division instructions
- maintaining an accumulator and a counter across multiple iterations
- conditional execution of instructions within the loop body
- calling a helper function from an assembly function

## Note

For these tasks the point is not for the solution to be "the fastest", but for you to manually trace how a C++ loop and simple logic translate into assembly steps.
