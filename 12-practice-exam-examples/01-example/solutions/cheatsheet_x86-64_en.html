x86-64 Cheatsheet

General Registers

64-bit registers

RAX, RBX, RCX, RDX, RDI, RSI, RBP, RSP, R8, R9, R10, ..., R15
General-purpose registers
RIP
Instruction pointer, the address of the current instruction

Register parts

  • EAX, AX, AL - lower 32/16/8 bits of RAX
  • EDI, DI, DIL - lower 32/16/8 bits of RDI
  • R8D, R8W, R8B - lower 32/16/8 bits of R8
  • RSP - stack pointer
  • RBP - base pointer for the current stack frame

Memory Access

Basic instructions

  • MOV RAX, [RBX] - load a 64-bit value from the address in RBX into RAX
  • MOV [RBX], EAX - store the 32-bit value from EAX to the address in RBX
  • LEA RAX, [RBX+4*RCX] - load the address RBX + 4*RCX into RAX

Addressing mode

[base + index*scale + offset]

  • base - base register, such as RAX or RBX
  • index - index register, cannot be RSP
  • scale - index multiplier: 1, 2, 4, or 8
  • offset - constant displacement

Arithmetic and Bitwise Operations

Arithmetic instructions

  • ADD RAX, RBX - addition: RAX = RAX + RBX
  • SUB RAX, RBX - subtraction: RAX = RAX - RBX
  • IDIV, DIV RDI - signed and unsigned division: RAX = RDX:RAX / RDI (quotient in RAX), RDX = RDX:RAX % RDI (remainder in RDX)
  • IMUL, MUL RDI - signed and unsigned multiplication: RDX:RAX = RAX * RDI
  • INC, DEC - increment or decrement by 1

Bitwise operations

  • AND, OR, XOR, NOT - bitwise operations
  • SHL, SHR, SAR, ROL, ROR - shifts and rotations

Flags (RFLAGS)

Important flags

CF
Unsigned overflow or borrow
PF
Parity
ZF
Zero
SF
Sign
OF
Signed overflow

Instructions that set flags

CMP (comparison), TEST (bitwise test), arithmetic and bitwise instructions such as ADD, SUB, and AND.

Jumps and Branching

Basic jumps

  • JMP label - unconditional jump
  • CALL label - function call
  • RET - return from function

Conditional jumps

  • JE/JZ, JNE/JNZ - equal/zero, not equal/not zero
  • JL, JG, JLE, JGE - signed comparisons
  • JB, JA, JBE, JAE - unsigned comparisons

Function Calling Convention (x86-64 ABI)

Argument and return-value registers

RDI
1st argument
RSI
2nd argument
RDX
3rd argument
RCX
4th argument
R8
5th argument
R9
6th argument

Saved registers

  • RAX, RCX, RDX, RSI, RDI, R8-R11 - caller-saved, may be clobbered by a call
  • RBX, RBP, R12-R15 - callee-saved, must be preserved by the called function
  • RAX - return value

Stack alignment

The stack must be 16-byte aligned before a function call, so RSP must be divisible by 16.

Function prologue

Manual:
PUSH RBP MOV RBP, RSP SUB RSP, <size>
Automatic:
ENTER <size>, 0

Function epilogue

Manual:
MOV RSP, RBP POP RBP RET
Automatic:
LEAVE RET

Compilation and Debugging

Compilation

g++ file1.cpp file2.s -g -o program

Running

./program

Debugging with GDB

gdb ./program

Debugger commands

  • run - run the program
  • b function_name - breakpoint at a function
  • b 0x400123 - breakpoint at address 0x400123
  • b file.cpp:10 - breakpoint at line 10 in file.cpp
  • next - next instruction, without entering functions
  • step - next instruction, entering functions
  • continue - continue to the next breakpoint
  • print RAX - show the value of register RAX
  • backtrace - show the function call stack
  • tui reg general - show registers in TUI mode
  • quit - exit the debugger

Most commands have short forms: b for break, c for continue, and so on.
