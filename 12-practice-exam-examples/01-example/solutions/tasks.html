Task 1 - x86-64

In the file 1.s, implement the function:

extern "C" int more_even_digits(long *a, int n);

The function receives an array a of nonnegative integers of length n. Return the number of elements whose decimal representation has strictly more even digits than odd digits.

The digit 0 is considered even. The number 0 has one digit, namely 0.

If n <= 0, the function returns 0.

The file 1.cpp reads n, then n numbers on one line, calls the function, and prints the return value.

Example 1

Input:

5
12 204 777 0 1357

Output:

2

Example 2

Input:

3
22 13 456

Output:

2

Example 3

Input:

1
7

Output:

0

Task 2 - x86-64

The following helper function is given in 2.cpp:

extern "C" void minmax(long *a, int n, long *pmin, long *pmax);

For an array a of length n > 0, minmax writes the smallest element to address pmin and the largest element to address pmax.

In the file 2.s, implement the function:

extern "C" int count_between_extremes(long *a, int n);

The function returns the number of array elements that are strictly greater than the minimum and strictly less than the maximum. The variables for the minimum and maximum must be local variables on the stack. Global variables are not allowed.

If n <= 0, the function returns 0.

The file 2.cpp reads n, then n numbers on one line, calls the function, and prints the return value.

Example 1

Input:

6
5 1 7 3 7 2

Output:

3

Example 2

Input:

5
10 20 30 40 50

Output:

3

Example 3

Input:

3
5 5 5

Output:

0

Task 3 - ARM64

In the file 3.s, implement the function:

extern "C" int clean_string(char *s);

The function modifies the string s in place and returns the number of changed characters.

Modification rules:

  • if the character is a digit from '0' to '9', replace it with '#';
  • if the character is an uppercase letter from 'A' to 'Z', convert it to the corresponding lowercase letter;
  • all other characters remain unchanged.

The string is terminated by the null character '\0'.

The file 3.cpp reads one line of text, calls the function, prints the modified string, and then prints the number of changed characters.

Example 1

Input:

A1 bZ!

Output:

a# bz!
3

Example 2

Input:

Hello123

Output:

hello###
4

Example 3

Input:

abcdef

Output:

abcdef
0

Task 4 - ARM64

The following helper function is given in 4.cpp:

extern "C" int is_colorful(long x);

A number is colorful if adjacent digits in its decimal representation always have different parity. A one-digit number is always colorful.

In the file 4.s, implement the function:

extern "C" long sum_colorful(long *a, int n);

The function goes through all elements of the array and calls is_colorful(a[i]) for each a[i] using the bl instruction. If is_colorful returns a nonzero value, a[i] is added to the sum. The function returns the obtained sum.

If n <= 0, the function returns 0. You may assume all elements are nonnegative.

The file 4.cpp reads n, then n numbers on one line, calls the function, and prints the return value.

Example 1

Input:

6
1234 1123 5050 248 7 135

Output:

6291

Example 2

Input:

3
1 2 3

Output:

6

Example 3

Input:

1
0

Output:

0
