Technical Details

Notes

The exam contains two x86-64 assembly tasks (tasks 1 and 2) and two ARM64 assembly tasks (tasks 3 and 4).

Task 1 - x86-64

In the file 1.s, implement the function:

extern "C" unsigned long weight(unsigned long x);

The weight of a nonnegative number is the sum of the squares of its decimal digits. The weight of 0 is 0.

The file 1.cpp reads the number x, calls the function, and prints the return value.

Example 1

Input:

203

Output:

13

Example 2

Input:

77

Output:

98

Example 3

Input:

0

Output:

0

Task 2 - x86-64

Use the function weight from the previous task. In the file 2.s, implement the function:

extern "C" unsigned long heaviest(unsigned long *a, int n);

The function receives an array a of length n. Find and return the element with the largest weight. The function weight must be called using the call instruction.

If several elements have the same largest weight, return the first such element in the array.

If n <= 0, the function returns 0.

The file 2.cpp reads n, then n numbers on one line, calls the function, and prints the return value. When compiling, also link the file 1.s.

Example 1

Input:

4
203 77 90 5

Output:

77

Example 2

Input:

3
1 9 3

Output:

9

Example 3

Input:

5
10 20 30 40 50

Output:

50

Task 3 - ARM64

In the file 3.s, implement the function:

extern "C" int count_in_range(unsigned long *a, int n,
                              unsigned long l, unsigned long r);

The function receives an array a of length n and two bounds l and r. Return the number of elements that belong to the closed interval [l, r].

You may assume l <= r.

If n <= 0, the function returns 0.

The file 3.cpp reads n, then n numbers on one line, then l and r on the same line, calls the function, and prints the return value.

Example 1

Input:

6
3 10 7 15 8 2
5 10

Output:

3

Example 2

Input:

4
1 2 3 4
2 3

Output:

2

Example 3

Input:

3
5 10 15
0 100

Output:

3

Task 4 - ARM64

The helper function statistics is given in 4.cpp, and count_in_range is the function from task 3:

extern "C" void statistics(unsigned long *a, int n,
                           unsigned long *pmean,
                           unsigned long *pmedian);

extern "C" int count_in_range(unsigned long *a, int n,
                              unsigned long l, unsigned long r);

For an array a of length n > 0, statistics writes the integer arithmetic mean to address pmean and the median to address pmedian.

In the file 4.s, implement the function:

extern "C" int count_typical(unsigned long *a, int n);

The function counts how many array elements are between the arithmetic mean and the median, including the bounds. Use the function count_in_range for counting. The variables for the mean and median must be local variables on the stack. Global variables are not allowed.

If n <= 0, the function returns 0.

The file 4.cpp reads n, then n numbers on one line, calls the function, and prints the return value. When compiling, also link the file 3.s.

Example 1

Input:

5
2 4 6 8 20

Output:

2

Example 2

Input:

4
1 3 5 7

Output:

1

Example 3

Input:

3
10 20 30

Output:

1
