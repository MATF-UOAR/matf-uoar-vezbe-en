# Arithmetic mean
In assembly, implement a function that computes the arithmetic mean of an array, along with a C program that calls it. The function's arguments are
a pointer to the first element of the array and the number of elements in the array.
## Example
```
5
1 2 3 4 5 
3
```
## Compilation
```
g++ main.cpp arithmetic_mean.s
```
## Running
```
./a.out
```
