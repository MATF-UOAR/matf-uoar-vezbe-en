# Square negatives
In assembly, implement a function that squares all negative elements of an array of integers, along with a C program that calls it. The
function's arguments are the address of the first element of the array and the number of elements.

## Example
```
6
3 4 -1 2 -5 3
3 4 1 2 25 3
```
## Compilation
```
g++ main.cpp square_negatives.s
```
## Running
```
./a.out
```
