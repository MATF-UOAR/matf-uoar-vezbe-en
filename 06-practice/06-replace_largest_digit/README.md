# Replace the largest digit
In assembly, implement a function that returns the number obtained when, in the number `n`,
all occurrences of its largest digit are replaced by the digit `c`, along with a C program that calls it.
You may use the function that finds the largest digit of a number (from one of the previous tasks) as a helper.
## Example
```
297693 4
247643
```
## Compilation
```
g++ main.cpp largest_digit.s replace_largest_digit.s
```
## Running
```
./a.out
```
