# Greater than the arithmetic mean
In assembly, implement a function that squares all elements of an array that are greater than its arithmetic mean, along with a C function that calls it.
You are allowed to use the function from the previous task to compute the arithmetic mean.

## Example
```
5
1 2 3 4 5
1 2 3 16 25
```
## Compilation
```
g++ main.cpp arithmetic_mean.s count_above_mean.s
```
## Running
```
./a.out
```
