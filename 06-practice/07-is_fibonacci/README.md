# Belongs to Fibonacci
In assembly, implement a function that checks whether a given number belongs to the Fibonacci sequence and, if it does, returns its index in that sequence.
If the number is 1, you should return 1. If the number does not belong to the sequence, you should return -1. You also need to write a C program that calls this function.

## Example
```
55
index: 10
```
## Compilation
```
g++ main.cpp is_fibonacci.s
```
## Running
```
./a.out
```
