# Exercises

This unit contains four short examples for independent work after the second week.

The idea is not to introduce new concepts, but to reinforce:

- `cmp` and conditional jumps
- several consecutive branches in a single function
- using the remainder of a division as a condition
- a simple `switch` with grouped `case` branches
- calling `puts` from an assembly function
