# Self-study mini examples

This unit contains three short examples for self-study after `Addition` and `Hello world`.

The idea is not to introduce completely new concepts, but to reinforce:

- the calling convention
- working with `eax`, `edi`, `esi`, `edx`
- the basic arithmetic instructions `add` and `sub`
- the difference between a value and an address
- calling the `puts` function
- the `.rodata` section and stack alignment before `call`
